# ⚡ webpack-boilerplate ver 0.0.2

express/babel/webpack을 모두 적용한 보일러플레이트입니다!  

별도의 템플릿 엔진을 사용하지 않고 html을 그대로 사용하였고, public 안에 dist가 존재해서 이 곳에 번들링 데이터가 들어가게 됩니다.

---

## 💥 ver 0.0.2 패치사항 :

- js파일이 수정되었습니다.

- 아기고양이 사진이 추가되었습니다.

- dist 폴더가 public 밖에서 public 안으로 이동했습니다.

---

## 🥕 번들링 데이터

.js, .scss, (.png,svg,jpg,jpeg,gif), (.woff,woff2,eot,ttf,otf), (csv,tsv), xml  

**※ 추가로 html도 dist에 들어가게 됩니다**

---

## 🍋 파일 구조

- .gitignore에 의해 node_modules와 dist 폴더는 포함되지 않습니다.
